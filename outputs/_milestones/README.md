# Milestones (Tracked)

This folder is the tracked window for milestone artifacts.
Everything else under `outputs/` is local and disposable.

Each milestone should be a minimal evidence pack:
- `summary.md`
- `key_metrics.json`
- `results_table.csv`
- `trades_head.csv`
- `state_events.csv` (if available)
- `effective_config.yaml`
- `figures/` (select 3-8 key images + compare/report if relevant)
- `repro/command.txt` and `repro/git_sha.txt`
