## 3.1 Momentum Trading（PDF p8）
- 价格梯度：`G_t(n) = P_t - P_{t-n+1}`。
- 加权移动平均：`A_t(n) = (∑_{i=0}^{n-1} w_i P_{t-i}) / (∑_{i=0}^{n-1} w_i)`。
- 买入信号：同时满足 `G_t(n) > 1` 且 `P_t - A_t(n) > 1`。

## 3.2 Reversion-to-Mean（PDF p8）
- 价格低于均值时倾向回归，作为买入信号。

## 3.3 Calculating Profitability（PDF p9）
- 动量交易盈利性：与 `gradient` 成正比。
- 均值回归盈利性：与 `(A_t - P_t)^2` 成正比。

## 3.4 Selling（PDF p9）
- 卖出条件：`S = (1 - f_a)P - (∑ (1 - f_{a_i})C_i)/n - L`，若 `S > 0` 则卖出。
- 强调平均成本（average price paid）。

## 3.5 Extreme Market Conditions（PDF p10）
- 触发条件：梯度超过历史平均梯度的 5 倍。
- 公式：`E = G_t(n) - C * (∑ G_t(i) / n)`，`C` 默认为 5。
- 触发后：仅允许极端卖出逻辑，直到价格回落到 `E*max_price` 或低于 `max_price - ΔP`。

## 3.6 No-Buy Condition（PDF p10）
- 极端卖出后进入 no-buy，原文描述为 “no-buy for a number of days”。

## 3.7 Holding Assets（PDF p10）
- 买入后 `z` 天内不允许卖出，`z` 对应参数 `T`。

## 3.8 Daily Buy/Sell Order（PDF p10）
- 日内顺序：先检查 no-buy / extreme；如 extreme 生效则跳过常规流程；否则先卖后买。

## 3.9 Noise Normalization（PDF p11）
- 权重因子：`W = C / t^2`（t 为天数）。
- 用途：降低噪声，比较 gold 与 bitcoin 时 BTC 权重更高。
